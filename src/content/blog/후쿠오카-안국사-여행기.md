# 도심속의 사찰 안국사

안국사는 텐진역에서 5분 거리에 있는 사찰이다. 사실 뭔가 계획하고 방문한 것은 아니고 숙소 바로 뒤에 안국사가 있길래 한번 방문해봤다.

안국사는 조동종 사찰이다. 한국에서는 흔한 종파가 아니어서 솔직히 아는 것은 묵조선을 수행한다는 것 정도만 알고 있다.

한창 사람들이 다닐 점심 즈음이었지만 경내는 매우 한산했다. 조계사는 평일에도 사람이 많았던 것 같은데 여기는 조용했다.

만덕전이 이 절의 메인 건물인거 같은데. 문이 닫혀있어 그냥 문 앞에서 슥 보고 말았다.

| <img src="/blog/images/후쿠오카-안국사-여행기/만덕전.jpg"  width="500" height="500"> |

|                                        만덕전                                        |

조계사는 엄청 큰 나무가 여러 그루 있었는데 여기는 작고 정갈한 나무가 군데군데 심어져 있었다.  
도심 속에 있다보니 자꾸 조계사랑 비교하며 보게 되더라.

| <img src="/blog/images/후쿠오카-안국사-여행기/나무3.jpg"  width="500" height="500"> |
| :---------------------------------------------------------------------------------: |
|                       가레산스이 스타일의 정원에 심어진 나무                        |

그렇게 슥 둘러보고 있었는데 커다란 범종 앞에서 기도하고 계신 스님을 발견했다. 기도라고 해야 하나, 수행이라고 해야 하나. 아무튼 계셨다.

| <img src="/blog/images/후쿠오카-안국사-여행기/안국사스님.jpg"  width="500" height="500"> |
| :--------------------------------------------------------------------------------------: |
|                        스님이 엄청 젊어보였다. (나보다도 어릴듯)                         |

나중에 알아본 바로는 매일 11시에 범종을 친다고 한다. 이것도 수행의 일환인 것 같다. 스님이 꽤나 창백하고 연약해 보였는데 괜찮으시려나 하는 걱정과 함께 기다렸다.

11시가 다 되어가니 사람들이 꽤 모였다. 종 치는거 보는게 하나의 커다란 볼거리라고 하더라.

안국사 바로 옆에는 후쿠오카 조리사 전문학교가 있다. 나도 한때는 진지하게 일본 전문학교에 갈 생각을 했었던지라 흐뭇한 마음으로 바라봤다.

| <img src="/blog/images/후쿠오카-안국사-여행기/후쿠오카조리사전문학교.jpg"  width="500" height="500"> |
| :--------------------------------------------------------------------------------------------------: |
|                                미래의 조리사들이 열심히 훈련 중이다.                                 |

잠시 기다리니 스님이 종을 치기 시작한다. 종을 한번 치고 경내가 떠나가라 뭔가 외친다. 종 소리가 근방에 은은히 퍼져서 아주 묘한 분위기를 풍긴다.

목소리가 아주 앳되보인다. 이걸 몇번이나 반복 하신다. 일정이 있어 계속 볼 수는 없었고 적당할 때 안국사를 나왔다.

나오고 한참을 걸어가도 종 소리가 들리니 참 기분이 좋았다.

# 텐진에 간다면 한번 가보면 좋을 것 같다.

하루에 딱 한번 오전 11시에 스님이 종을 울리니 그 때 맞춰서 방문하면 좋을 것 같다.  
종 소리가 마음을 차분하게 한다. 도심속의 사찰은 언제나 매력적이다.
